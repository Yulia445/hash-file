#  Simple Password Guard

Цей код демонструє, як можна безпечно зберігати паролі за допомогою **хешування**

###  Що цей скрипт вміє?

1. **Реєстрація:** Якщо файл порожній, скрипт бере ваш пароль, «зашифровує» його (створює хеш) і зберігає у файл `brat.txt`.
2. **Перевірка:** Якщо у файлі вже є хеш, скрипт порівнює введений вами пароль із тим, що збережено.


###  Як це працює 

Програма використовує бібліотеку **bcrypt**. На відміну від звичайного шифрування, хешування — це «дорога в один бік». Ми не розшифровуємо пароль назад, ми просто створюємо складний «відбиток» пароля.

**Логіка роботи:**

* **`readFileSync`**: Читаємо вміст файлу.
* **`hashSync`**: Перетворюємо ваш пароль, наприклад `12345`, на щось страшне на кшталт `$2b$10$X...`.
* **`compareSync`**: Магічна функція, яка перевіряє, чи ваш ввід відповідає збереженому хешу.


###  Як запустити у себе

1. **Встановіть залежності:**
Вам знадобиться пакет `bcrypt`. Відкрийте термінал і напишіть:
```bash
npm install bcrypt

```


2. **Запустіть скрипт вперше (реєстрація пароля):**
Замість `мій_пароль` напишіть свій варіант.
```bash
node script.js мій_пароль

```


3. **Спробуйте увійти:**
Запустіть команду ще раз. Якщо пароль правильний — отримаєте доступ!
